# 🚀 প্রজেক্ট লাইভ করার পূর্ণাঙ্গ নির্দেশিকা

আপনি একদম নতুন হলেও ভয়ের কিছু নেই। নিচের ধাপগুলো একটি একটি করে অনুসরণ করুন। আমি আপনার হয়ে সব ফাইল এবং সেটিংস রেডি করে দিয়েছি।

---

## ধাপ ১: GitHub-এ কোড তোলা (GitHub Setup)

১. প্রথমে আপনার ব্রাউজারে [GitHub.com](https://github.com/)-এ যান এবং সাইন ইন করুন।
২. একটি নতুন রিপোজিটরি (New Repository) তৈরি করুন। নাম দিন `hrms-live` (বা আপনার পছন্দমতো)।
৩. রিপোজিটরি তৈরি করার পর আপনি কিছু কোড দেখতে পাবেন। সেখান থেকে **"Push an existing repository..."** সেকশনের কমান্ডগুলো কপি করুন।
৪. আপনার পিসির টার্মিনালের নিচের কমান্ডগুলো একে একে টাইপ করুন:

```bash
# সব ফাইল অ্যাড করুন
git add .

# প্রথম কমিট করুন
git commit -m "Initialize deployment files"

# মেইন ব্রাঞ্চ সেট করুন
git branch -M main

# আপনার GitHub লিঙ্কটি এখানে দিন (নিচের লিঙ্কটি আপনার GitHub থেকে কপি করা লিঙ্ক দিয়ে রিপ্লেস করুন)
git remote add origin https://github.com/আপনার-ইউজার-নেম/hrms-live.git

# কোড পুশ করুন
git push -u origin main
```

## ধাপ ২: Render-এ অ্যাকাউন্ট খোলা

১. [Render.com](https://render.com/)-এ যান এবং GitHub একাউন্ট দিয়ে লগইন করুন।
২. আপনার ইমেইল ভেরিফাই করুন।

---

## ধাপ ৩: অটোমেটিক ডিপ্লয়মেন্ট (Render Blueprint)

আমি আপনার প্রজেক্টে একটি `render.yaml` ফাইল দিয়েছি যা আপনার হয়ে অটোমেটিক ডাটাবেস এবং ওয়েব সার্ভার তৈরি করে দেবে।

১. Render ড্যাশবোর্ডে গিয়ে **"Blueprints"** লিঙ্কে ক্লিক করুন।
২. **"New Blueprint Instance"** বাটনে ক্লিক করুন।
৩. আপনার GitHub রিপোজিটরি (`hrms-live`) সিলেক্ট করুন।
৪. Render আপনাকে কিছু সেটিংস দেখাবে। সব ঠিক থাকলে **"Apply"** বাটনে ক্লিক করুন।

Render এখন আপনার জন্য:
- একটি **PostgreSQL** ডাটাবেস তৈরি করবে।
- আপনার Laravel অ্যাপটি তৈরি (Build) করবে।
- ডাটাবেস মাইগ্রেশন চালাবে।

---

## ধাপ ৪: এনভায়রনমেন্ট ভ্যারিয়েবল (Environment Variables)

Render-এ অ্যাপটি তৈরি হয়ে গেলে, **Environment** ট্যাবে গিয়ে চেক করুন সব ঠিক আছে কিনা। বিশেষ করে:
- `APP_KEY` (যদি না থাকে তবে একটি তৈরি করে দিতে হবে, আমি অটোমেটিক জেনারেট হওয়ার সিস্টেম রেখেছি)।

- **ডাটাবেস:** প্রথমবার অ্যাপটি চালু হতে ৫-১০ মিনিট সময় নিতে পারে। একটু ধৈর্য ধরুন।
- **এডিট:** আপনি যখনই কোড এডিট করে GitHub-এ পুশ করবেন, Render সাথে সাথে তা অটোমেটিক লাইভ করে দেবে।
